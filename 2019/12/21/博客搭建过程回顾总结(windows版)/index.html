    博客搭建过程回顾总结(windows版)

    安装nodejs

    官方网址:(https://nodejs.org/en/)

    成功验证: 安装完成后点击win+r,输入cmd打开窗口,输入node -v后回车.
    安装成功后会显示node的版本信息.

    安装git

    官方网址:https://gitforwindows.org/

    成功验证: 同上,在窗口内输入git后回车.

    注册github账号

    官网:https://github.com/

    设置github的SSH keys

    git安装完成后,打开安装目录,找到git-bash.exe,双击.
    在打开的窗口中输入:ssh-keygen -t rsa -C "github邮箱",回车(回车到无变化).点X退出.
    找到id_rsa.pub文件(通常在电脑的用户的.ssh文件夹内,如:C:\Users\ASUS.ssh),用写字板打开,全选(ctrl+a)复制(ctrl+c).

    登陆github账号.依次点击:
    右上头像->Settings->SSH and GPG keys->New SSH key
    title框填写用户名;
    Key框内粘贴(crtl+v)上一步复制的内容.

    使用hexo

    网址:https://hexo.io/
    复制搜索框内的代码(npm install hexo-cli -g),粘贴到cmd命令打开的窗口中,回车.

    成功验证:在cmd打开的窗口中输入hexo v.显示版本信息.
    完成后.
    输入npm install hexo-deployer-git --save回车.

    生成初始博客

    在任意一个盘(如D盘)中新建一个文件夹,命名为blog.
    在文件夹上面的的路径栏(例如:此电脑 > 本地磁盘(D:) > blog)内,输入cmd弹出命令窗口.
    窗口前缀如: D:\blog>_
    在窗口中输入:hexo init. //(初始化)生成初始的博客.
    输入:npm install. //安装依赖库
    输入:hexo s. //运行测试,浏览器打开地址，http://localhost:4000/

    用hexo将博客部署到github上

    进入github网页,新建一个仓库.即:
    右上角的+号->New repository
    Repository name格式为:Owner(账户名).github.io
    (必须严格按此格式,否则后面可能会404!)
    ->Create repository
    复制网址栏网址(如https://github.com/itermyang/itermyang.github.io).

    修改博客的配置文件

    打开blog文件夹.
    用文本编辑器(如vscode)打开_config.yml.
    将第16行url修改为上一步复制的网址,删掉中间的两端.如:
    url: https://itermyang.github.io
    (即你到时直接访问的网址.)

    ctrl+s保存

    部署到githab命令

    在cmd命令窗口中接着输入:
    git config --global user.email "github注册邮箱"回车
    git config --global user.name "github用户名"回车

    hexo g == hexo generate //生成一套静态网页
    hexo s == hexo server //启动服务预览
    hexo d == hexo deploy //部署

    ps: 在命令行输入密码时先复制‘token’文件(在.ssh文件夹中)；点击鼠标右键(粘贴);回车

    在浏览器的网址栏输入网址(url),如https://itermyang.github.io

    跟换主题

    代码: git clone https://github.com/yelog/hexo-theme-3-hexo.git themes/3-hexo
    解析: git clone 中意的github的主题的网址.git themes/主题名
    将_config.yml文件中第96行theme的默认值(landscape)修改为文件名.
    如:theme: 3-hexo
